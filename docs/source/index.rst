¡Bienvenido a la documentación de ctrutils!
==========================================

.. admonition:: **⚠️ Estado del Proyecto**
   :class: warning

   Este proyecto está **bajo desarrollo** y puede estar sujeto a cambios frecuentes. ¡Gracias por tu paciencia!

ctrutils es una librería de utilidades en Python creada para simplificar y automatizar tareas comunes en ciencia de datos y desarrollo. Con un conjunto de herramientas integradas, ctrutils permite una mayor eficiencia en el manejo de datos, conexión a bases de datos, y más.

Repositorio del Proyecto
------------------------
El código fuente está disponible en GitHub. Contribuciones, issues y sugerencias son bienvenidas:

- **GitHub**: `CristianTacoronteRivero/ctutils <https://github.com/CristianTacoronteRivero/ctutils.git>`_

Instalación
-----------
Para instalar ctrutils, utiliza `pip`:

.. code-block:: bash

   $ pip install ctrutils

Navegación por la Documentación
-------------------------------
La documentación está organizada de la siguiente forma:

- **Guía de Instalación**: Cómo instalar y configurar ctrutils.
- **Documentación de API**: Descripción detallada de las funciones y clases de la librería.
